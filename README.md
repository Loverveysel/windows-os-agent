# Windows OS Agent

## Overview
The Windows OS Agent is designed to securely execute JSON commands generated by a language model (LLM) using a Planner-Executor architecture. The agent takes user input, generates commands, and executes them in a controlled environment, ensuring safety and compliance with defined security policies.

## Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   cd windows-os-agent
   ```

2. Install dependencies:
   - Use the provided PowerShell script:
     ```
     .\scripts\installer.ps1
     ```
   - Alternatively, install manually using pip:
     ```
     pip install -r requirements.txt
     ```

## Usage
1. Run the agent:
   ```
   python src/agent/main.py
   ```

2. Follow the prompts to input your request. The agent will generate a JSON command and execute it securely.

## Security
The agent implements strict security policies to ensure that only safe commands are executed. All commands are validated against predefined schemas and executed in a sandboxed environment to prevent potential harm.

## Testing
Unit tests are provided for each module. To run the tests, use:
```
pytest tests/
```
